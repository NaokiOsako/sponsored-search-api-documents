

# ReportJobStatus (enum)

A status of the requested report.

#### Service

+ [ReportDefinitionService](../../services/ReportDefinitionService.md)

#### Namespace

[ReportDefinitionService#Namespace](../../services/ReportDefinitionService.md#namespace)

| Enumeration  |       Type       |          Description          |
| ------------ | ---------------- | ----------------------------- |
| WAIT | xsd:string | Please wait for report request to complete. |
| COMPLETED | xsd:string | Report request completed successfully. |
| IN_PROGRESS | xsd:string | Report is in creating process. |
| FAILED | xsd:string | Report request failed. |
