# EnhancedCpcEnabled (enum)
EnhancedCpcEnabled specifies flag of Maximize Conversions.

### Service
+ [CampaignService](../../services/CampaignService.md)

### Namespace
[CampaignService#Namespace](../../services/CampaignService.md#namespace)

| Enumeration | Type | Description |
|---|---|---|
| FALSE | xsd:string | Invalidate the setting of Maximize Conversions |
| TRUE | xsd:string | Activate the setting of Maximize Conversions |
