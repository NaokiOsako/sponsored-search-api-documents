# RaiseBidWhenBudgetConstrained
RaiseBidWhenBudgetConstrained is a PAGE_ONE_PROMOTED strategy that can specify if its allowed to raise bid when the budget is constrained.

### Service
+ [BiddingStrategyService](../../services/BiddingStrategyService.md)

### Namespace
[BiddingStrategyService#Namespace](../../services/BiddingStrategyService.md#namespace)

| Enumeration | Description | 
|---|---|
| ACTIVE| Adjust to raise the bid, even if the budget constraint	. |
| DEACTIVE| Will not adjust to raise the bid, if the budget constraint. |
