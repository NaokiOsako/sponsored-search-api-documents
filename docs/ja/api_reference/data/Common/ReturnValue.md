# ReturnValue
ReturnValueオブジェクトは、処理の結果を表します。
### Service
全サービス共通
### Namespace
http://ss.yahooapis.jp/V201805

| フィールド | データ型 | 説明 |
|---|---|---|
| operationSucceeded | xsd:boolean | 処理結果です。<br>trueの場合は、処理は成功しました。<br>falseの場合は処理が失敗しています。 |
| error[] | <a href="../Common/Error.md">Error</a> | エラー内容です。 |
