# DeliveryStatus (enum)
DeliveryStatusは、アカウントの配信状況を表します。

### Service
+ [AccountService](../../services/AccountService.md)

### Namespace
[AccountService#Namespace](../../services/AccountService.md#namespace)

| 値 | データ型 | 説明 |
|---|---|---|
| ACTIVE| string| 配信状況はオンです。 |
| PAUSED| string| 配信状況はオフです。 |
